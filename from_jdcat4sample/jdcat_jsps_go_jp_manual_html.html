JDCat利用マニュアル

検索を行う

「トップ」タブをクリックすると、キーワード検索欄、ファセット検索欄が表示されます。これらを用いて公開中のデータを検索することができます。

キーワード検索を行う

ここでは、キーワードを指定して、アイテムを検索する方法手順を説明します。キーワード検索には、簡易検索と詳細検索があります。

簡易検索

簡易検索とは、キーワード検索テキストボックスに入力した文字列が、登録されているデータの様々な情報（タイトル、作成者等）のどこかに合致したデータを探し出します。
項目説明
キーワード検索欄
データを検索するためのキーワードを入力します。
空白文字（半角・全角空白）区切りで複数のキーワードを入力できます。この場合はＡＮＤ検索となります。
・以下の文字を含めた検索が可能です

+ - = && || ! ( ) { } [ ] ^ " ~="" *="" ?="" :="" \=""


・「<」「>」を含めた検索はできません。
・数値をキーワードとする場合は全角・半角文字のどちらでもかまいません。
検索方式ラジオボタン・全文にチェックを入れ検索すると、登録されたメタデータと登録されたファイルの内容から検索した結果を表示します。
・キーワードにチェックを入れ検索すると、 登録されたメタデータから検索した結果を表示します。
検索ボタンクリックすると、キーワードで簡易検索を開始します。
検索結果の詳細については、検索結果画面についてを参照してください。
詳細検索ボタンクリックすると、［検索条件入力］画面が表示されます。詳細については、詳細検索を参照してください。

＊入力したキーワードのクリアボタンはありません。トップタブをクリックすると、検索語をクリアしたトップ画面に戻ります。

  • 検索例
    「家族」「日本」「2011」を含む研究データを検索する場合、下記のように入力し、検索ボタンを押します。

詳細検索

複数の検索条件とキーワードを指定してアイテムを検索します。指定した各検索項目のAND条件で検索を行います。詳細検索ボタンをクリックすることで、詳細検索画面が表示されます。

項目説明
検索条件リスト検索条件をプルダウンリストから選択します。
JDCatの項目のうち、以下の項目を設定できます。
タイトル、作成者、配布者、トピック、概要、対象地域、助成機関、研究費番号、関連研究、関連文献、編集者、所蔵者・寄託者、対象時期
・これらの項目の詳細はJDCatメタデータスキーマを参照してください。
検索キーワード入力欄データを検索するためのキーワードを入力します。
対象時期はyyyymmdd形式にしたがって半角数字で入力します。なおmmとddは省略できます。
例：2021年6月と入力する：202106
検索条件削除ボタンクリックすると、検索条件から検索条件を削除できます。
検索条件追加ボタンクリックすると、検索条件を追加できます。
追加した検索条件の項目は検索条件リストから選択してください
＊初期状態では、すべての項目が表示されているので。新たな項目を追加できません。
検索ボタンクリックすると、検索が開始されます。
検索結果の詳細については、検索結果画面についてを参照してください。
クリアボタンクリックすると、初期検索条件に戻ります。
＊簡易検索のテキストボックスはクリアされません。

ファセットで検索する

ここでは、ファセットを使った絞り込み検索について説明します。
ファセット検索とは、JDCatがあらかじめ用意した検索項目を選ぶことによって、検索対象を絞り込むナビゲーションシステムです。JDCatでは、トピック、アクセス権、配布者、データタイプ、対象地域、対象時期、データの言語の7つの項目について、セレクトボックス、チェックボックスリスト、レンジスライダーを利用して絞り込み検索を行うことができます。

トピックデータの内容を表すキーワード。
CESSDA Topic Classification、日本統計年鑑目次、日本十進分類表で登録されています。		セレクトボックス
アクセス権データセットのアクセス状態（利用制限など）に関する情報。
詳しくはアクセス権の統制語彙を参照してください。		チェックボックスリスト
配布者データを頒布する機関。
現時点では拠点機関が登録されています。		チェックボックスリスト
データタイプ収集されたデータのタイプ。詳しくはデータタイプの統制語彙を参照してください。チェックボックスリスト
対象地域収集されたデータが対象とする地理的範囲。
詳しくは対象地域の統制語彙を参照してください。		セレクトボックス
対象時期収集されたデータが対象とする時間的範囲。ファセット検索では、年単位で設定できます。
レンジスライダー
データの言語データ内で用いられている主な言語。
詳しくはデータの言語の統制語彙を参照してください。		チェックボックスリスト

セレクトボックスを利用したファセット検索

トピックと対象地域の検索は次のように操作します。


項目説明
ファセット開閉ボタン初期状態では、各ファセット検索は開いています。ファセット開閉ボタンをクリックすることで、キーワード入力欄とファセット検索ボタンを利用することができます。
ファセット検索ボタン

ファセット検索で利用可能なキーワードとその表示件数がリストで表示されます。
このリストの中から絞り込みに利用したいキーワードをクリックすると、検索キーワードとして設定することができます。
このキーワードは複数選択することができます。
＊キーワードの表示件数はファセットで絞り込む前のものが表示されております。表示件数はファセットで絞り込んだ後も変化しません。
キーワード入力欄

ファセット検索で利用可能なキーワードを検索することができます。入力に合わせて検索を行い、絞り込んだリストの中から検索に利用したいキーワードをクリックすることで、検索キーワードとして設定することができます。

チェックボックスリストを利用したファセット検索

アクセス権、配布者、データタイプ、データの言語の検索は次のように操作します。


項目説明
ファセット開閉ボタン初期状態では、各ファセット検索は開いています。ファセット開閉ボタンをクリックすることで、キーワード入力欄とファセット検索ボタンを利用することができます。
チェックリストボックスファセット検索で利用可能なキーワードとその表示件数がリストで表示されます。
このリストの中から絞り込みに利用したいキーワード左側のチェックボックスをクリックすると、検索キーワードとして設定することができます。
このキーワードは複数チェックすることができます。
＊キーワードの表示件数はファセットで絞り込む前のものが表示されております。表示件数はファセットで絞り込んだ後も変化しません。

レンジスライダーを利用したファセット検索

対象時期は次のように操作します。


項目説明
開始年対象時期の開始年を設定することができます。半角数値で入力することができます。
終了年対象時期の終了年を設定することができます。半角数値で入力することができます。
スライダー両端の円を左右にドラッグすることで、開始年と終了年を設定することができます。
移動ボタンクリックすると、設定した開始年と終了年で絞り込み検索を行うことができます。
*絞り込み検索の結果に合わせて、開始年と終了年が再設定されるため、検索時に設定した開始年、終了年と異なることがあります
*絞り込み検索後、スライダーの両端は開始年と終了年に設定されます。検索範囲を広げる際は開始年、終了年を入力してください。

検索結果画面について

ここでは、検索結果画面の見かたについて説明します。



簡易検索、詳細検索、ファセット検索を行うと、画面の右下に検索結果が表示されます。検索結果は、アイテムという単位で整理され、表示されます。

項目説明
表示順アイテムの表示順を下記の項目から選択し、変更することができます。
Titleタイトル名によって並び替えることができます。
Relevance関連度の高い順に並び替えることができます。（降順のみ。昇順への並び替えはできません）
Temporal対象時期によって並び替えることができます。
asc昇順に並び替えます。
desc降順に並び替えます。
表示数1ページあたりのアイテム表示数を下記の項目から選択し、変更することができます。
表示数は20, 50, 75, 100の4種類から選択できます。
タイトルクリックすると、詳細画面に遷移します。
メタ情報当該データのメタ情報が表示されます。ここでは作成者、配布機関、概要情報が表示されます。
＊一部データは作成者が日英併記で表示されています

詳細画面について

ここでは、詳細画面の見かたについて説明します。
検索結果画面に表示されている各アイテムのタイトルをクリックすると詳細画面が表示されます。社会科学分野と人文学分野では、表示内容は異なっています。
各項目は原則として多言語併記で表記されますが、言語の順番は拠点機関によって異なります。

社会科学分野の詳細画面

社会科学分野の研究データは、次のような形式で表示されます。



項目説明
タイトル研究データの名前が表示されます。
公開日タイトルで示されたデータがJDCatに登録された日が表示されます。当該のデータが各機関でいつ公開されたかは、各機関にてご確認ください。
URLデータのランディングページが表示されています。
＊DOIの取得にはタイムラグがあるため、反映されていないことがあります。
利用統計当該詳細画面の閲覧数が確認できます。下記の項目を利用することで1ヶ月単位での閲覧数が確認できます。
期間プルダウンリストから期間を１ヶ月単位で選択することができます。
選択した期間ごとの閲覧数が表示されます。
See detailsクリックすると、期間で設定した月のおける閲覧数を確認することができます。
バージョンJDCatに登録されたアイテムのバージョン情報が表示されます。当該データのバージョン情報はアイテム内の情報を参照してください。
バージョン
バージョンの表示フォーマットは、「yyyy-mm-dd hh:mm:ss.999999」の形式で表示されます。
Show ALL versionsクリックすると、すべてのバージョンが表示されます。
エクスポートアイテムのメタデータをJPCOAR形式、DublinCore形式、DDI形式で出力することができます。
